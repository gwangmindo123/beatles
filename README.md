# 🎸 THE BEATLES UNIVERSE
***웹으로 만나는 비틀즈의 역사 · 음악 · 감성***

비틀즈의 명곡, 이야기, 그리고 오늘의 기분까지 책임지는 올인원 웹사이트

## 🎯 프로젝트 개요 (The Project Mission)
이 웹사이트는 전설적인 밴드 '비틀즈'를 기리고, 팬들에게 그들의 음악 세계를 깊이 있게 탐험할 수 있는 인터랙티브한 플랫폼을 제공하기 위해 제작되었습니다. 비틀즈의 역사, 멤버, 주요 앨범 및 명곡에 대한 정보를 제공하며, 퀴즈와 음악 추천 기능을 통해 사용자들이 비틀즈와 더욱 즐겁게 소통할 수 있도록 돕는 것을 목표로 합니다.

## 🛠️ 사용 기술 (Technologies Used)

- **HTML5**: 웹 페이지의 구조를 위해 사용.
- **CSS3**: 스타일링 및 레이아웃을 위해 사용.
- **JavaScript (ES6+)**: 인터랙티브 기능 및 동적 콘텐츠를 위해 사용.
- **jQuery**: 특히 퀴즈 및 기타 인터랙티브 요소에서 DOM 조작 및 이벤트 처리를 위해 사용.

## 🚀 주요 기능 (Key Features)

이 웹사이트는 네 가지 주요 페이지로 구성되어 있습니다:

### 1. 🏠 홈 (Home)
메인 페이지는 방문자들에게 비틀즈를 소개합니다. 다음 내용을 포함합니다:
- **역사**: 밴드의 결성과 여정 이야기.
- **상징적인 앨범**: 가장 영향력 있는 앨범 소개.
- **멤버**: 존 레논, 폴 매카트니, 조지 해리슨, 링고 스타의 프로필.
- **유명한 노래 & 유산**: 음악사에 미친 영향력 개요.

### 2. ❓ 퀴즈 (Quiz)
인터랙티브 퀴즈로 비틀즈에 대한 당신의 지식을 시험해보세요! 이 페이지는 가장 헌신적인 팬들조차도 도전하게 만드는 동적으로 로드되는 질문들을 특징으로 합니다. 홈페이지의 내용을 꼼꼼히 탐색했다면 퀴즈를 쉽게 풀 수 있을 것입니다.

### 3. 🎵 음악 추천 (Music Recommendations)
무엇을 들어야 할지 모르겠나요? 이 페이지는 당신의 현재 기분에 따라 비틀즈 노래를 추천합니다. 새로운 애창곡을 발견하거나 오래된 클래식을 다시 방문하는 재미있는 방법입니다.

### 4. 🎶 음악 감상 (Music appreciation)
비틀즈의 세계에 빠져보세요. 이 페이지에서는 엄선된 그들의 최고 히트곡들을 감상할 수 있습니다.
